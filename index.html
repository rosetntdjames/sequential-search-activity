03
Problem Solving · Back-Calculation

A sequential search is run on an unknown array. You are told:

  • Each comparison takes 5 ms.
  • An average-case search (target always present, uniform distribution) completed in 27.5 ms.
  • A subsequent worst-case search (target absent) was also timed.
  • (a) How many average-case comparisons were made?
  • (b) Solve for n: how many elements are in the array?
  • (c) How long does the worst-case search take (in ms)?
04
Derivation · Average Case

For a sequential search on an array of n elements where the target is always present and equally likely to be at any position, the average number of comparisons is given by the formula:

Cavg = (n + 1) / 2

  • (a) Apply the formula: what is the average number of comparisons when n = 7?
  • (b) Apply the formula: what is the average number of comparisons when n = 15?
  • (c) Why is the average-case complexity still O(n)?
05
Problem Solving · Solve for n

A sequential search runs on an unknown array. You are given the following facts:

  • Each comparison takes 6 ms.
  • A worst-case search (target not present) completed in 72 ms.
  • A best-case search (target at index 0) costs the minimum possible time.
  • (a) How many elements are in the array? (solve for n)
  • (b) How long does the best-case search take (in ms)?
  • (c) What is the maximum number of elements the array can have if the worst-case search must stay strictly under 54 ms?
